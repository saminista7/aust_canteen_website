# AUST Canteen Management Website

A full-stack canteen management web application developed to digitize food ordering and administrative workflows for a university canteen. The system combines a modern, interactive frontend with database-driven backend operations.

This project was developed as part of an undergraduate software development coursework.

---

## Overview
The AUST Canteen Management Website allows users to browse food items, place orders, and interact with the canteen system through a responsive and dynamic web interface. Administrators can manage food items, categories, and orders through a dedicated admin panel.

The application emphasizes usability, real-time interaction, and structured data handling.

---

## Key Features
- Landing page with modern UI and animations
- Category-based food browsing
- Food listing and ordering system
- Live search functionality using AJAX
- Contact page
- Admin dashboard for managing food items and orders
- Dynamic UI updates without full page reloads
